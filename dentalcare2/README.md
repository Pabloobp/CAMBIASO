# DentalCare

Aplicación Flutter para gestión clínica dental con panel para cliente y dentista.

## Funcionalidades

- Autenticación dual cliente/dentista
- Persistencia de sesión
- Registro con validaciones
- Gestión de citas con estados y confirmaciones
- Historial de tratamientos
- Configuración de servicios y disponibilidad
- Base de datos local SQLite con datos demo

## Credenciales demo

- Dentista: `dentista@dentalcare.demo` / `Dental123`
- Cliente: `cliente@dentalcare.demo` / `Dental123`

## Ejecución

```bash
flutter pub get
flutter run
```
